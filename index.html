Meerbot is a Raspberry Pi based three wheel robot. It is powered by 3V motors that allow it to traverse through space. Python library gpio_zero is used to control the motors. The bot is controlled over wifi with a built-in R/Shiny server.

The name Meerbot stands for the Bionic Meerkat.
